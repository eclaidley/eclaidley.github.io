Patatap Clone

A clone of Patatap, built with the Paper.js and howler.js libraries, based on an exercise in Colt Steele's "The Web Developer Bootcamp" course.

Click here to give it a try.

How It Works

  • Each letter key from A - Z is assigned to a corresponding sound event and color animation.
  • Press any letter key(s) to trigger these events.
